Permet d'afficher une ou plusieurs vues partielles Affiche une vue si $nomVue est une chaine de charactère ou affiche plusieurs vues si $nomVue est un tableau contentant le nom des vues. Affiche toujours la/les vue(s) entre les vues partielles entete.php et piedPage.php.

Parameters
[string/array]$nomVue Nom de la vue ou tableau contentant les noms des vues à afficher
[array]$donnees Données passée à la/aux vues
[boolean]$complete Vrai = entête, vue(s) & pied de page / Faux = vue(s) seulement
Returns
message d'erreur ou une vue
